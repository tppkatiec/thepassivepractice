# Logo — drop files here

## Ideal uploads (any you have)

| Filename | Why |
|---|---|
| `logo.svg` | Primary — vector, scales anywhere |
| `logo-mark.svg` | Just the icon/monogram (no wordmark) |
| `logo-wordmark.svg` | Just the wordmark (no icon) |
| `logo-light.svg` / `logo-dark.svg` | Variants for light vs. dark backgrounds |
| `logo@2x.png` | Raster fallback, ~1200px wide |
| `favicon.svg` (or `.ico`) | Browser tab |

## Color guidance
- The brand canvas is purple (`#6772c3`), so the **primary logo should read on purple**.
- If your master is dark-on-light, also include an inverted (`logo-light.svg`) that works on the bg.
